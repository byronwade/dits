## How to Contribute

We welcome contributions from everyone! Please see our [Contributing Guide](docs/development/contributing.md) for details on how to get started.

### Types of Contributions

- **Code**: Bug fixes, new features, performance improvements
- **Documentation**: Tutorials, guides, API documentation
- **Testing**: Additional test cases, bug reports
- **Design**: UI/UX improvements, logo, branding
- **Community**: Answering questions, helping new users
